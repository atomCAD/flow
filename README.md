# Flow

A modern scientific workflow language written in Rust, designed for building robust and type-safe data processing pipelines.

## Overview

Flow is a new scientific workflow language that combines the power of Rust's type system with container-based process isolation. It provides a declarative way to define data processing pipelines while ensuring type safety and runtime isolation.

### Key Features

- **Rust-like Syntax**: Familiar and ergonomic syntax for Rust developers
- **Strong Type System**: Leverages Rust's type system for compile-time guarantees
- **Container Isolation**: Each process runs in its own container (Docker/Podman)
- **Channel-based Dataflow**: Efficient data transfer between processes
- **WUFFS Integration**: Safe and efficient serialization/deserialization
- **Modern Design**: Built with modern software engineering principles

## Project Status

This project is in early development. The design and implementation are being actively worked on.
